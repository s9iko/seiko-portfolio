## Overview

SneakHaven is a comprehensive sneaker resell store UI mockup designed in Adobe Illustrator. The design includes all essential e-commerce pages: homepage, product catalog, product details, checkout, user account, login, and about page.

## Key Pages Designed

**Homepage:**
- Hero banner with "STAY TUNED" messaging and release date (12.09)
- Exclusive Collections carousel featuring sneakers (Air Jordan, Air Max, Field General, etc.)
- Collaborations section with video content from artists (Drake, Tyler Creator, Central Cee, Travis Scott)
- Sport category tiles (Basketball, Running, Skateboard, Golfing)

**Navigation:**
- Side menu with New Arrivals, Sales & Offers, Upcoming Releases, Limited Editions, Brands, Collaborations

**Product Pages:**
- Grid layout showing sneaker catalog with product images, names, and colorways
- Detailed product page with image gallery, descriptions, size selector, reviews (888 reviews, 4.5 stars), and Add to Cart

**Shopping Experience:**
- Order History showing past purchases with product details and prices
- Checkout page with cart summary, delivery calculations, voucher code input, and product recommendations

**Additional Pages:**
- Login page with email/password authentication and social login options
- "WE ARE SNEAKHAVEN" brand story page explaining the company's mission and values

## Design Features

- Clean, modern interface optimized for browsing sneakers
- Product-focused layout with high-quality sneaker photography
- Organized navigation for easy category filtering
- Complete user flow from browsing to checkout
- Consistent branding with SneakHaven logo throughout

## Technologies Used

- **Design Tool**: Adobe Illustrator
- **Deliverables**: High-fidelity UI mockups for complete e-commerce flow

## Outcome

This project demonstrates the ability to design a complete e-commerce platform from homepage to checkout, with attention to user flow, product presentation, and brand identity. The mockup serves as a comprehensive design guide for a sneaker resell platform.
