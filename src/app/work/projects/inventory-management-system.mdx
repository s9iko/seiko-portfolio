## Overview

A lightweight Windows Forms app that lets small teams manage products with fast CRUD, multi-filter search, and built-in validation. It keeps data clean, prevents duplicates, and calculates totals automatically.

## Highlights
- Manage products (add / edit / delete) with supplier, date, and pricing details
- Live filtering by supplier, date range, and price—great for quick lookups
- Automatic total amount (Qty × Price) and guardrails to block bad data
- DataGridView UX for quick edits plus safe multi-delete with confirmation
- Parameterized queries everywhere to avoid SQL injection

## Stack
- Windows Forms (C#)
- ADO.NET (SqlClient) on SQL Server LocalDB
- .NET Framework 4.7.2

## Outcome
Delivered a working desktop app that keeps inventory accurate, searches fast, and data safe—solid practice in WinForms, SQL Server, and defensive data handling.
